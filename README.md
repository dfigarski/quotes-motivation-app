# Motivational Quotes App

## Opis projektu
Aplikacja Angular do przeglądania motywacyjnych cytatów. Projekt będzie rozwijany o integrację z Quotable API, ranking cytatów oraz dark mode.

## Temat projektu
Aplikacja motywacyjnych cytatów z użyciem Quotable API.

## Opis domeny
Aplikacja służy do wyświetlania i oceniania motywacyjnych cytatów w prostym interfejsie.

## Użytkownik
Osoba zainteresowana cytatami motywacyjnymi.

## Problem
Brak prostego miejsca do czytelnego przeglądania cytatów i oceniania ich.

## Główne encje
- cytat
- autor
- ocena
- ulubiony cytat

## Planowane operacje CRUD
- Create
- Read
- Update
- Delete

## Technologie
- Angular
- TypeScript
- SCSS
- Angular Router
